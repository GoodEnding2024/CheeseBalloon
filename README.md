# CheeseBalloon

## 프로젝트 소개

<p align="justify">
치즈벌룬은 국내 인터넷 방송 서비스 숲(SOOP)과 치지직(CHZZK)의 방송인 통계를 제공하는 서비스입니다.
실시간 방송중인 목록과 간단한 통계 정보를 통해 인터넷 방송 팬들이 보다 취미를 즐길 수 있도록 돕습니다.
</p>

<br>
